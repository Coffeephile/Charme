apl_request

Info

Make a request to a charme Server

Params

requestDatajsonData in json format
callbackfunctionCallback function
SessionIdstringNot used currently, Write "null" in here
ServerstringServer to which the request is send to. If its null, we will use the host of the logged in user.

Example Code

  // Send a info_about request to myserver.com
  apl_request({"requests" :
    [
      {"id" : "info_about"}
    ]
  }, function(data){ console.log(data.info_about);}, null, "myserver.com");

apl_user

Info

Returns a User Object.

Params

uidStringThe user id, "me@myserver.com" for example.

Example Code

var user = apl_user("me@myserver.com");
console.log(user.server); // myserver.com
console.log(user.username); // me
console.log(user.userIdURL); // me@myserver.com encoded with encodeURIComponent for URL requests.
